# bogy-music-vis

Artistic music visualization with PyGame


## Setup

- `py -m venv .venv`
- Windows: `.venv\Scripts\activate`, Other: `source .venv/bin/activate`
- `pip install -r .\requirements.txt`
- If pygame install has issues: `py -m pip install -U pygame`


## Start Game

- `py game.py`
